# Silent Orbit Public Beta Testing

Silent Orbit is a local-first bilingual atlas for finding, understanding, and navigating AI Skills across Systems and Libraries.

Live demo: https://silent-orbit-skills-library.netlify.app/

## Seven beta tasks

1. Describe what this product is in one sentence before reading further documentation.
2. Search for one concrete goal you actually care about.
3. Open one Skill from the results.
4. Explain how System, Library, and Skill differ in this product.
5. Switch between English and Chinese and note anything that becomes unclear.
6. Complete one core path on a phone or narrow mobile viewport.
7. Record the most confusing part, the part you liked most, and the thing you most wanted to click.

## Severity

- **P0:** privacy leak, site unavailable, incorrect public data, or a failed core flow.
- **P1:** important functionality cannot be discovered, mobile use is blocked, or the interface is materially misleading.
- **P2:** non-blocking detail, copy, or visual problem.
- **Idea:** a future suggestion rather than a defect in the beta promise.

## Privacy boundary

The site has no third-party analytics, cookies, or behavior tracking. Outcome records and language preferences stay in the current browser's local storage. Do not include secrets, private prompts, private memories, machine paths, or personal account information in GitHub feedback.

Safari is an external beta check because Phase 4A did not have a local Safari runtime. Please include the browser name, version, device, and viewport when reporting a Safari result.

Use the [beta feedback template](./BETA_FEEDBACK_TEMPLATE.md), the Bug Report issue form, or the Experience Feedback issue form in this repository.
